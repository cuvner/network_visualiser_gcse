GCSE focus

  • A LAN is a local area network inside one site, like a home or school.
  • A WAN links LANs over a large area, usually using the internet.
  • Routers move data between networks. Switches connect wired devices inside a LAN.
  • Wireless access points let laptops and phones join without cables.
